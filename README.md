## Minesweeper

This is a simple demo of the game Minesweeper written in Flutter for the web!

[Click here to play the game!](https://minesweeper-32e4c.firebaseapp.com/#/)

## To build it yourself

Update packages:

```flutter pub get```

Run in Chrome:

```flutter run -d chrome```
